Welcome to CADLab, the cadquery customizer.

Select an assembly on the left.

Then select the parameters you want to change and click Customize.

That will show you the updated assembly.

You can then either view each individual model, or download the STL files for the models.

You can fork this project and populate the assemblies folder with your own parametric models and deploy this on heroku (or anywhere else) to host your own models.
